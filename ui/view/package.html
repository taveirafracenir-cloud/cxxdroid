CXXDROID UI View Package

The ui.view package is the core of the CXXDROID graphical interface. It provides the essential classes and components for building interactive, animated, and accessible user interfaces.

Package Structure

  • animations/ - Contains all UI animations like Fade and Slide.
  • touch/ - Handles touch input through Input and TouchEvent.
  • Accessibility/ - Manages accessibility features like TalkBack.
  • webkit/ - Provides a WebView component for displaying HTML content.
  • Menu - Interactive menu system.
  • View - Core class managing child views, layout, rendering, input, and accessibility.

Main Classes

View

The main class representing a UI element. Supports:

  • Child views hierarchy
  • Layout and rendering
  • Animations (Fade, Slide)
  • Touch input handling
  • Accessibility / TalkBack integration
  • Menu and WebView components
  • LITHIUM sensor detection
View root = new View("root");
root.initialize();
root.layout();
root.render();

Menu

Interactive menu for navigating UI sections.

Menu menu = new Menu();
menu.addItem("Home");
menu.addItem("Settings");
menu.display();

WebView

Displays HTML content in the UI. Supports:

  • Loading HTML files
  • Rendering content inside the View hierarchy
WebView webView = new WebView();
webView.loadUrl("example.html");
webView.display();

Input & TouchEvent

Handles touch input and events.

Input input = new Input();
input.addEvent(new Input.TouchEvent(100,200, Input.TouchEvent.Type.TAP));
input.processEvents();

Animations

Provide animated transitions for Views:

  • Fade - fade in/out UI elements
  • Slide - move elements smoothly across the screen
Fade fade = new Fade(1000, true); // 1 second fade-in
view.startAnimation(fade);

Slide slide = new Slide(800,0,0,100,0); // slide menu
view.startAnimation(slide);

Accessibility

Supports TalkBack and other accessibility features:

view.enableAccessibility();
view.getTalkBack().speak("Clicked on Home");

Usage Example

View root = new View("root");
root.initialize();
root.enableAccessibility();

root.getInputHandler().addEvent(new Input.TouchEvent(10,10,Input.TouchEvent.Type.TAP));
root.processInput();

root.menu.display();
root.webView.loadUrl("example.html");
root.render();

CXXDROID UI View package provides the foundation for building modern, interactive, and accessible desktop interfaces.
